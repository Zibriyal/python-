# 🐍 Python Learning Repository

Welcome to the **Python Learning Repository** — a structured collection of Python fundamentals and examples designed to help learners understand the core concepts of Python programming.  

> 🚀 **Note:** This project is still under development. More topics, examples, and advanced exercises will be added soon!

---

## 📁 Repository Structure

Below is an overview of the folders included in this repository:

| Folder | Description |
|--------|--------------|
| **Comparison operators** | Learn how to compare values in Python using equality, inequality, and relational operators. |
| **Flow control** | Understand how to control the flow of a Python program using conditionals and branching logic. |
| **Functions** | Explore Python functions, parameters, return values, and scope. |
| **Hello Wolrd!** | Your first Python program — learn basic syntax and printing. |
| **Input and Output** | Manage user input and display program output effectively. |
| **List** | Dive into Python lists — creation, manipulation, and common operations. |
| **Loops** | Master looping constructs such as `for` and `while` loops. |
| **Object Oriented Programming with Python** | Introduction to OOP principles — classes, objects, inheritance, and more. |
| **Operators** | Overview of Python’s arithmetic, logical, and bitwise operators. |
| **Python data types** | Learn about Python’s core data types — integers, floats, strings, and booleans. |
| **String** | Work with strings: slicing, formatting, and manipulation techniques. |
| **Python object and data structure basics** | Foundational knowledge of Python’s built-in data structures and object behavior. |
